Installing vim bindings

General commands

The following commands are always available.

State change commands:

i       Change state to insert from command state
Ctrl-[  Change state to command from insert state
Esc     Change state to command from insert state

Miscellaneous commands:

:       Enter vi's command line
Ctrl-:  Enter Leo's command line
Ctrl-s  Save '.leo' file
u       Undo previous command
Ctrl-r  Redo previous command
Tab     Toggle focus between Outline and Body pane
=       Simulate double-click on current node's icon box
Alt-G   Go to specified line number (relative to derived file)

Outline commands

Outline navigation and manipulation are mapped to keys using vi's text navigation key mapping as a guide. Outline navigation commands can also be invoked while in the body pane's command mode. This reduces the keystrokes necessary to make changes to the body text of multiple nodes. Node headline editing uses vi's line editing key mappings.

The following commands are supported when in a headline's command mode.

State change commands:

Ctrl-i  Change state to command from grayed state
return  Change state to command from insert state
Ctrl-]  Change state to grayed from command state

Go to node commands:

Ctrl-h  Go to parent
Ctrl-j  Go to previous sibling
Ctrl-k  Go to next sigling
Ctrl-l  Go to first child, expanding the present node if needed.
Ctrl-G  Go to the outline's last node
gg      Go to the outline's first node

Note: The bindings above can also be used while in the command state of the node's body.

Scroll commands:

Ctrl-y  Scroll outline up one line   (also Ctrl-b, Ctrl-u)
Ctrl-e  Scroll outline down one line (also Ctrl-f, Ctrl-d)

Move node commands:

Shift-Ctrl-h    Move node left
Shift-Ctrl-l    Move node right
Shift-Ctrl-j    Move node up
Shift-Ctrl-k    Move node down

Mark node commands:

m       Mark or unmark the current node
Ctrl-m  Go to next marked node
Alt-m   Clear all marked nodes

Clone (transclude) node commands:

t       Clone node
Ctrl-t  Go to next cloned node

Select headline text:

s       Select current character
W       Select text to the end of the current word
B       Select text to the beginning of the current word
Shift-D Select to end of line

At present, the delete, yank, and change text object commands are not available. Instead, commands are provided to facilitate the selection of text which the user can follow with 'i', 'x', or 'y' to change, delete, or yank the text.

Delete/Yank/Paste node commands:

Ctrl-x  Delete current node
Ctrl-c  Yank current node
Ctrl-v  Paste current node

Move cursor comamnds:

w   Go to next word
b   Go to previous word
e   Go to endo of word
|   Go to Beginning of line
$   Go to end of line

Find commands:

f   Find character

Body pane

All vi text navigation and editing functionality has been implemented except for the following.

The following vi commands are not supported in the Body pane:

cf<char>    Change up to and including specified character
ct<char>    Change up to specified character
yf<char>    Yank up to and including specified character
yt<char>    Yank up to specified character
t<char>     Move cursor to character before specified character
r           Replace a single character with a single character
?           Search backwards
1G          Use 'gg' instead
0           Use '-' instead
bksp        Move one character to the left
m<label>    Mark text
`           Mark text
'           Goto marked text
~           Toggle character's case
.           Repeat last command
<n><cmd>    Perform command 'n' number of times
<cmd><n><object>    Performing the command on/upto n'th object
Ctrl-u      Move cursor up  1/2 screen
Ctrl-d      Move cursor down 1/2 screen
Ctrl-e      Slide buffer up (leaving cursor on same line)
Ctrl-y      Slide buffer down (leaving cursor on same line)
z           Slide buffer to put current line at top
z.          Slide buffer to put current line at middle
z-          Slide buffer to put current line at bottom
